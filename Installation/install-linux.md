# Installing Git on Linux

Git may already be installed on your Linux system. To check, open a terminal and type:

```bash
git --version
If Git is installed, you will see the version number, e.g.:

git version 2.42.0
If Git is not installed, follow the steps for your Linux distribution.

Ubuntu / Debian
Update package lists:

sudo apt update
Install Git:

sudo apt install git
Verify installation:

git --version
Fedora / CentOS / RHEL
Fedora:

sudo dnf install git
CentOS / RHEL:

sudo yum install git
Verify installation:

git --version
Arch Linux
Install Git:

sudo pacman -S git
Verify installation:

git --version
✅ Notes
After installation, you can use Git commands in the terminal for local repositories and syncing with GitHub.

Updating Git is easier if you use your distribution’s package manager.
